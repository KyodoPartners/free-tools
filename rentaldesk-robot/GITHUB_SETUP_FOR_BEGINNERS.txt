PUTTING RENTALDESK ROBOT ON GITHUB (AND GETTING A FREE LIVE WEBSITE)
====================================================================
No command line needed. You do all of this in your web browser.
Total time: about 10 minutes.

WHAT YOU HAVE
-------------
A folder called "rentaldesk-robot" with these files inside:
  index.html, styles.css, app.js, README.md, LICENSE,
  GITHUB_SETUP_FOR_BEGINNERS.txt
Keep them all together in that one folder.

BEFORE YOU START
----------------
1. Open the LICENSE file and replace "YOUR NAME HERE" with your name.
2. (Optional) You can test the app right now: double-click index.html
   and it opens in your browser. If it works on your computer, it will
   work online.


STEP 1 - MAKE A FREE GITHUB ACCOUNT
-----------------------------------
- Go to https://github.com and click "Sign up". It's free.
- Pick a username you're happy to show employers (it becomes part of
  your live web address), e.g. "lad-durham".


STEP 2 - CREATE A NEW REPOSITORY (a "repo" = a project folder online)
---------------------------------------------------------------------
- Click the "+" in the top-right corner, then "New repository".
- Repository name: rentaldesk-robot   (lowercase, no spaces)
- Description: "Free open-source landlord dashboard (browser app)"
- Choose "Public" (so employers can see it).
- Do NOT check "Add a README" (you already have one).
- Click "Create repository".


STEP 3 - UPLOAD YOUR FILES (drag and drop)
------------------------------------------
- On the new empty repo page, click the link
  "uploading an existing file".
- Open your "rentaldesk-robot" folder on your computer.
- Select ALL the files inside it (Ctrl+A on Windows/Linux, Cmd+A on Mac).
- Drag them into the GitHub upload area in your browser.
- Wait for them to finish uploading.
- In the "Commit changes" box, type: "Initial upload"
- Click "Commit changes".

That's it - your code is now on GitHub. You can stop here if you only
wanted the code online. To get a clickable LIVE website, do Step 4.


STEP 4 - TURN ON THE FREE LIVE WEBSITE (GitHub Pages)
-----------------------------------------------------
- In your repo, click "Settings" (top menu).
- In the left menu, click "Pages".
- Under "Build and deployment" -> "Source", choose "Deploy from a branch".
- Set Branch to "main" and folder to "/ (root)". Click "Save".
- Wait 1-2 minutes, then refresh the page. GitHub shows a green box with
  your live address, which looks like:
      https://YOUR-USERNAME.github.io/rentaldesk-robot/
- Click it. Your app is now live on the internet, for free.


STEP 5 - POLISH IT FOR EMPLOYERS
--------------------------------
- Edit the README on GitHub (click README.md, then the pencil icon) and
  replace "YOUR-USERNAME" in the live-demo line with your real username.
- Put that live link on your resume / LinkedIn:
  "RentalDesk Robot - a working browser app I built. [live demo] [code]"
- Bonus: take a screenshot of the dashboard and drag it into the README
  so people see it without clicking.


HOW TO MAKE CHANGES LATER
-------------------------
- Go to the file on GitHub, click the pencil (edit) icon, make your change,
  and click "Commit changes". The live website updates automatically in
  a minute or two.


TROUBLESHOOTING
---------------
- "The live page is blank / 404": wait 2 minutes and refresh; make sure
  index.html is in the top level of the repo (not inside a sub-folder).
- "Excel button does nothing": you need to be online the first time so the
  Excel library can load; after that give it a second and click again.
- "My data disappeared": data is saved per-browser. A different browser or
  a cleared cache starts fresh. Use "Export to Excel" to keep a backup.
